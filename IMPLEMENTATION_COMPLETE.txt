================================================================================
                            HOW IT WORKS
================================================================================

1. HTML page loads with <body data-page="pagename">
2. JavaScript reads data-page attribute
3. JavaScript fetches /data/pagename.json
4. JavaScript parses JSON structure
5. JavaScript renders content dynamically
6. User sees populated page

FALLBACK: If JSON fails to load, page displays static HTML content

================================================================================
                      ADDING A NEW PAGE
================================================================================

1. Create HTML file with data-page attribute:
   <body data-page="newpage">

2. Create JSON file at /data/newpage.json:
   {
     "hero": { "title": "...", "subtitle": "..." },
     "sections": [ ... ]
   }

3. Done! JavaScript automatically loads and renders the data

================================================================================
                      UPDATING CONTENT
================================================================================

To update any page's content:

1. Edit the corresponding JSON file in /data/
2. Refresh the page in browser
3. Changes appear immediately

Example: Update home page title
  Edit: /data/home.json
  Change: "title": "New Title"
  Result: Appears on /

================================================================================
                      SUPPORTED JSON CONTENT
================================================================================

Paragraph:
  { "type": "paragraph", "text": "Your text here" }

Unordered List:
  { "type": "list", "items": ["Item 1", "Item 2"] }

Ordered List:
  { "type": "list", "ordered": true, "items": ["Step 1", "Step 2"] }

Article (with title):
  { "title": "Article Title", "content": "Text or content array" }

Section:
  { "title": "Section Title", "content": [...], "articles": [...] }
