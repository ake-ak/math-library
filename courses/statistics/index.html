Statistics

The science of data: collecting, analyzing, and interpreting information to make decisions.

← Back to Courses | Math Path
Active Course

Overview

Statistics is the science of learning from data. In an age of information, understanding statistics—from data collection to inference and decision-making—is essential. This course develops both the theoretical foundations and practical skills needed to work with data responsibly and effectively.

Prerequisites

Basic familiarity with algebra and functions. Some comfort with mathematical notation is helpful.

Learning Objectives

  • Understand probability and probability distributions
  • Master descriptive statistics and data visualization
  • Understand statistical inference and hypothesis testing
  • Work with confidence intervals and estimation
  • Understand regression and correlation
  • Apply statistics to real-world problems

Course Modules

Module 1: Probability Foundations

The mathematical foundation of statistics. Understanding probability, random variables, and probability distributions.

  • Basic probability concepts and rules
  • Conditional probability and Bayes' theorem
  • Random variables and distributions
  • Expected value and variance

Module 2: Common Probability Distributions

Important probability distributions that appear throughout statistics and applications: normal, binomial, exponential, and others.

  • Discrete distributions (binomial, Poisson)
  • Continuous distributions (normal, exponential, uniform)
  • Properties and applications of distributions
  • The Central Limit Theorem

Module 3: Descriptive Statistics

Methods for summarizing and visualizing data. Understanding your data is the first step in any statistical analysis.

  • Measures of central tendency (mean, median, mode)
  • Measures of dispersion (variance, standard deviation)
  • Data visualization (histograms, box plots, scatter plots)
  • Exploratory data analysis

Module 4: Estimation and Confidence Intervals

Using sample data to estimate population parameters and quantify uncertainty.

  • Point estimation
  • Confidence intervals
  • Sampling distributions
  • Standard error and margin of error

Module 5: Hypothesis Testing

A systematic framework for testing claims about populations using sample data.

  • Hypotheses and test statistics
  • P-values and significance levels
  • Type I and Type II errors
  • One-sample and two-sample tests

Module 6: Regression and Correlation

Understanding relationships between variables and building predictive models.

  • Correlation and association
  • Simple linear regression
  • Multiple regression
  • Assumptions and diagnostics

References

Recommended books and resources:

  • Devore - Probability and Statistics for Engineering and the Sciences
  • Rice - Mathematical Statistics and Data Analysis
  • Freedman, Pisani, Purves - Statistics (accessible introduction)
