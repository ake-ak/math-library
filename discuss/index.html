Discussion & Community

Connect with other mathematicians, ask questions, and collaborate with the learning community.

Ways to Participate

We foster an active community of mathematicians and learners. Here are the primary ways to participate in discussions:

GitHub Discussions & Issues

Our main community forum is hosted on GitHub. This is the best place for:

  • Detailed discussions about course materials
  • Questions about specific mathematical topics
  • Sharing solutions and insights
  • Reporting issues or proposing improvements
  • Contributing ideas for new content

You'll need a free GitHub account to participate.

Visit GitHub Discussions

Community Guidelines

We are committed to maintaining a respectful, inclusive community. When participating, please:

  • Be respectful: Treat other community members with courtesy and respect, even in disagreement.
  • Be constructive: Focus on understanding and improving content and knowledge, not on personal criticism.
  • Be clear: Write clearly and provide context for your questions. Include relevant course or topic information.
  • Be open-minded: Mathematics is a collaborative endeavor. Be open to correction and different perspectives.
  • Search before asking: Check if your question has already been discussed to avoid duplication.
  • Respect intellectual property: When sharing solutions or resources, respect copyright and attribution.

Getting Help

Different questions deserve different approaches:

Stuck on a Problem?

Post in GitHub Discussions with details about what you've tried so far. The community can provide hints and guidance to help you work through it.

Conceptual Confusion?

Describe what you understand and where you get stuck. Often talking through the concept helps clarify it.

Found an Error?

Report it on GitHub Issues. Include the specific page and the error you found so we can fix it promptly.

Want to Discuss Mathematics?

Start a discussion topic in GitHub Discussions or email us if it's something more suited to private conversation.
